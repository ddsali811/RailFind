# RailFind 民眾端頁面草圖｜吳紜瑜

## 1. 搜尋頁

```text
┌─────────────────────────────────────────────┐
│ RailFind  鐵道遺失物跨站整合回報平台        │
├─────────────────────────────────────────────┤
│ 標題：輸入關鍵字，快速找到可能的遺失物       │
│ 說明：可輸入物品名稱、顏色、站點或特徵       │
│                                             │
│ [關鍵字輸入框：黑色雨傘                  ] [搜尋]
│                                             │
│ 類別篩選 [全部]  狀態篩選 [全部]  站點 [全部]
│                                             │
│ 快速測試：黑色雨傘｜藍色皮夾｜白色水壺       │
└─────────────────────────────────────────────┘
```

### 頁面重點

- 讓使用者一進來就知道要輸入什麼。
- 保留進階篩選，但不強迫填寫。
- 空白搜尋會顯示錯誤提示。

## 2. 結果列表頁

```text
┌─────────────────────────────────────────────┐
│ 搜尋結果：「黑色雨傘」共找到 3 筆可能結果    │
├───────────────┬───────────────┬─────────────┤
│ 92%           │ 68%           │ 51%         │
│ 黑色折疊雨傘  │ 深灰長柄雨傘  │ 黑色水壺    │
│ 北捷信義安和站│ 台鐵台中站    │ 桃捷A8站    │
│ 待認領        │ 待認領        │ 已領回      │
│ [查看詳細資料]│ [查看詳細資料]│ [查看詳細資料]
└───────────────┴───────────────┴─────────────┘
```

### 頁面重點

- 顯示相似度百分比，並依高到低排序。
- 每張卡片顯示物品名稱、狀態、站點、拾獲日期與匹配原因。
- 沒有結果時，不留空白，顯示「找不到符合條件的遺失物」。

```text
┌─────────────────────────────────────────────┐
│ ← 返回搜尋結果                              │
│                                             │
│ [物品圖示/照片]   物品編號 RF001            │

│ 類別：雨傘         顏色：黑色                │
│                                             │
│ 物品描述：黑色折疊雨傘，附傘套……            │
│                                             │
│ [聯絡保管站點] [繼續搜尋]                   │
└─────────────────────────────────────────────┘
```

### 頁面重點

- 提供使用者確認物品的完整資訊。
- 顯示聯絡保管站點的按鈕。
- 找不到物品 ID 時，顯示錯誤提示並引導回搜尋頁。
